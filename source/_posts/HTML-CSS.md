> 熟悉前端是我一直以来的理想，有时间就看下总结下。。。

```
<html>

<head>
<style type="text/css">
h1 {color: red}
p {color: blue}
</style>
</head>

<body>
<h1>header 1</h1>
<p>A paragraph.</p>
</body>

</html>
```

小白时期不了解HTML与CSS是怎么配合的，看上面代码 `<head>`标签中的CSS 声明`h1`颜色为红 `p` 颜色为蓝，其实就是把各个标签的样式在这个里面做一个总的约束，不用写到各个标签当中了。 提高了效率。
