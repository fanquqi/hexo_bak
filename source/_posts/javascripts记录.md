## 基本语法

- 赋值 
```
var a = 1;
```

- if 格式(if (条件) {执行语句} 支持嵌套)
```
if (2>1) {
    x = 1;
    y = 2;
    if (x>y) {
        result = 0;
    }
    if (x<y) {
        result = 1;
    }
}
```

- 注释
```
/* 这是注释 */
```
